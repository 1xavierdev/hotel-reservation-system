# Hotel Reservation System

A Java console application for managing hotel reservations with file upload/download capabilities and statistical analysis.

## Overview
Built as a final project for CS coursework (Fall 2023), this comprehensive hotel management system allows users to upload reservations from files, add/update reservations, view all bookings, and generate detailed statistics including sorted guest lists and total costs.

## Features
- **Upload Reservations**: Load multiple reservations from a text file
- **Add New Reservations**: Manually create new guest reservations
- **Update Reservations**: Modify existing reservation details by guest name
- **View All Reservations**: Display complete list of current reservations with costs
- **Download Statistics**: Generate stats file with sorted guest names, longest stay, and total revenue
- **Print Statistics**: View saved statistics from file

## Technologies
- Java
- File I/O (Scanner, FileWriter, PrintWriter)
- ArrayList data structure
- Console-based interface
